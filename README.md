# tmux-no-pane-cycle

Navigating between panes without cycle

## Installation with Tmux Plugin Manager (recommended)

Add plugin to the list of TPM plugins:

```tmux.conf
set -g @plugin 'tphiepbk/tmux-no-pane-cycle'
```

Use `prefix` + <kbd>I</kbd> to install it.
